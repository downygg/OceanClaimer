# How To use
- ```git clone https://github.com/downygg/OceanClaimer.git```
- ```cd OceanClaimer```
- ```npm i```
- ```cd v1.1.0```
- ```edit source.json base on wallet import format (secret key / mnemonic)```
- ```node index.js```
